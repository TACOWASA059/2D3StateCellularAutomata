# 2D3StateCellularAutomata
2次元3状態セルオートマトン
## コマンド
### 範囲指定
- /position1 - 範囲指定pos1
- /position2 - 範囲指定pos2 
### 初期化
- /initialize
### ランダマイズ

平面をランダム化する
- /randomize
### 生成
- /timeevolution <time_step>

time_stepは繰り返し回数
